# **App Name**: RojXOStream

## Core Features:

- Multi-Media Library: Browse and access a categorized library of anime, movies, songs, and YouTube shorts.
- Detailed Content Information: Display comprehensive details for each media item including movie/singer, character info for songs; writer/director, 4-5 sentence summary, moral, IMDb and Rotten Tomatoes ratings for anime/movies; and unique descriptions for YouTube shorts.
- Adaptive Media Playback: Enable full video playback for visual content and offer an audio-only listening mode specifically for songs. Display YouTube view counts for all relevant media.
- Immersive Shorts Feed: A dedicated, vertically scrollable feed for YouTube shorts, mimicking the native short-form video experience.
- AI-Powered Content Summarizer: Automatically generate concise 4-5 sentence summaries and derive a moral for anime and movies, assisting the user in populating media details. This AI tool analyzes provided text or metadata.
- Content Upload and Management: An administrative interface for the owner to upload or link media files and manage their metadata (descriptions, ratings, etc.) directly from local device or cloud sources.
- Metadata Storage: Securely store all media metadata, including descriptions, ratings, and links, in a scalable database.

## Style Guidelines:

- Dark color scheme for an immersive, cinematic viewing experience that reduces eye strain and emphasizes media content.
- Primary color (rich indigo-violet): #5D20DF, providing a sophisticated, imaginative, and modern digital aesthetic.
- Background color (dark, subtly violet-tinged): #231E30, serving as a deep, unobtrusive canvas that harmonizes with the primary color.
- Accent color (vibrant electric blue): #31ACFA, used for interactive elements, highlights, and calls to action, ensuring high contrast and a contemporary feel.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif), conveying a modern, digital, and slightly futuristic tone.
- Body text and detailed descriptions will utilize 'Inter' (sans-serif), chosen for its excellent readability, clean aesthetic, and objective clarity across various media information.
- Use sleek, minimalist line icons that align with the modern digital theme, ensuring clarity and unobtrusive design within the dark UI.
- Adopt a fluid, responsive layout with generous negative space, emphasizing media thumbnails and player components while maintaining intuitive navigation across different content types.
- Overall aesthetic: Maintain a minimalistic, clean, and professional design, focusing on content and user experience without unnecessary clutter.
- Dynamic Backgrounds: The background of the interface will change dynamically based on the currently playing content, displaying relevant scene photos or artwork for an immersive experience.
- Implement subtle transitions and loading animations, like fading content in or smooth scrolling, to enhance user engagement without distraction, particularly during media playback and shorts browsing.
